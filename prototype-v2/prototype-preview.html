📐 设计规范说明

设计风格：ardot 风格，简洁现代，注重可用性和用户体验

色彩体系：主色调 #2B5F75（深蓝），辅助色 #C73E3A（红色），背景色 #F5F0E8（米白）

字体规范：PingFang SC / Microsoft YaHei，标题 20-32px，正文 14-16px，辅助文字 12-13px

圆角规范：小圆角 8-10px，中圆角 15-20px，大圆角 20px+

阴影规范：轻度阴影 0 2px 12px rgba(0,0,0,0.06)，中度阴影 0 4px 20px rgba(0,0,0,0.1)，重度阴影 0 20px 60px rgba(0,0,0,0.15)

组件状态：所有交互组件均包含默认、悬停、点击、禁用、加载、空状态、错误状态等完整状态
