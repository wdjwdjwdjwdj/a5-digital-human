### 关键技术点
- `batch_edit` 操作：父节点先创建 → 用返回的 ID 引用创建子节点
- 颜色格式：SOLID fill 用 {r,g,b} 0-1 浮点数；GRADIENT_LINEAR 用 gradientStops(含a) + gradientTransform
- 字体处理：半粗体用 "Semi Bold" → Ardot 自动修正为 "SemiBold"
- 绝对定位子元素：在 flex 父级中需设 `layoutPositioning: "ABSOLUTE"`
- 毛玻璃效果：BACKGROUND_BLUR + 半透明 SOLID fill + stroke
