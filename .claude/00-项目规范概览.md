## 技术栈

| 模块 | 选型 | 版本 / 参数 |
|------|------|------------|
| 核心框架 | FastAPI + Uvicorn | Python ≥3.10, FastAPI ≥0.110 |
| LLM | DeepSeek API + 通义千问（降级） | `deepseek-chat`, api.deepseek.com/v1 |
| RAG 知识库 | Dify 社区版（首选）/ langchain+faiss（本地降级） | Docker 部署 / 纯 CPU 内存检索 |
| ASR | FunASR + Web Speech API（降级） | `paraformer-zh`, device: cpu |
| TTS | Edge-TTS + pyttsx3（降级） | `zh-CN-XiaoxiaoNeural`, ≥6.0 |
| **数字人** | **Three.js + @pixiv/three-vrm** | **r170 / v3.2.0, VRM 1.0 BlendShape** |
| 管理后台 | Streamlit | ≥1.35 |
| 数据库 | SQLite（WAL 模式） | Python 内置 |

> 变更：Live2D Cubism SDK → Three.js+@pixiv/three-vrm, 原因: 3D迁移
> 变更：Open-LLM-VTuber 框架 → FastAPI + Uvicorn（纯自建）
> 变更：Docker 部署 Dify 增加本地 faiss 降级方案

## 配置（.env 关键项）

```bash
# ===== LLM 配置 =====
DEEPSEEK_API_KEY=sk-your-key-here
DEEPSEEK_BASE_URL=https://api.deepseek.com/v1
DEEPSEEK_MODEL=deepseek-chat

# ===== 降级 LLM 配置（新增） =====
FALLBACK_LLM_API_KEY=your-key-here  # 通义千问 API Key

# ===== Dify 配置 =====
DIFY_API_URL=http://localhost/v1
DIFY_API_KEY=app-your-key-here

# ===== 语音配置 =====
TTS_PROVIDER=edge-tts
TTS_VOICE=zh-CN-XiaoxiaoNeural
ASR_PROVIDER=funasr

# ===== 3D VRM 数字人 =====
VRM_MODEL_URL=frontend/static/vrm/AliciaSolid.vrm

# ===== 管理后台 =====
ADMIN_PASSWORD=your-key-here        # ⚠️ 必须设置，不允许默认值
```

> conf.yaml 为配置参考文档，实际运行配置从 .env 通过 pydantic-settings 加载。

## 功能边界（关键变更）

| 编号 | 功能 | 变更 |
|------|------|------|
| F8 | 数字人配置管理 | 切换 **VRM 3D** 模型路径 + TTS 音色 |
| F9 | 口型同步 | **VRM BlendShape** 驱动口型同步 |

## 排除项

- ❌ 2D Live2D 渲染（已迁移至 3D VRM 方案）
- ❌ GPU 推理（Wav2Lip / SadTalker / LivePortrait）
- ❌ 前端框架（Vue / React），沿用原生 HTML/CSS/JS + ESM importmap

## 降级方案

| 主方案 | 降级到 | 切换条件 |
|--------|--------|---------|
| Three.js VRM | 静态 PNG + CSS 口型动画 | VRM 模型加载失败 |
| Edge-TTS | pyttsx3（离线 TTS） | 网络异常或 API 403 |
| FunASR | Web Speech API（浏览器内置） | 模型加载失败或超时 > 10s |
| DeepSeek API | 通义千问 API | API 连续 2 次 5xx（滑动窗口）|
| Dify RAG | local_rag（langchain+faiss）→ 直连 DeepSeek | Dify 不可用 |
