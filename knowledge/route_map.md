# 西湖景区路线指引

## 经典游览路线

### 半日游路线A（3-4小时，湖东精华线）
断桥残雪 → 白堤 → 平湖秋月 → 孤山（西泠印社）→ 岳王庙 → 曲院风荷 → 苏堤春晓 → 花港观鱼 → 雷峰塔
> 适合时间紧张、体力较好的游客

### 半日游路线B（3-4小时，湖西文化线）
灵隐寺（飞来峰造像）→ 龙井村 → 九溪烟树
> 适合对佛教文化和茶文化感兴趣的游客

### 一日游路线（6-8小时，全景环湖线）
断桥残雪 → 白堤 → 孤山（浙江省博物馆、西泠印社）→ 岳王庙 → 曲院风荷 → 苏堤（六桥）→ 花港观鱼 → 雷峰塔 → 三潭印月（游船登岛）→ 南屏晚钟（净慈寺）
> 覆盖西湖十景中的大部分，建议租共享单车辅助

### 二日游路线

#### 第一天（西湖经典环湖）
上午：断桥残雪 → 白堤 → 平湖秋月 → 孤山（西泠印社、浙江省博物馆）
下午：岳王庙 → 曲院风荷 → 苏堤 → 花港观鱼 → 雷峰塔 → 净慈寺（南屏晚钟）
晚上：湖滨银泰用餐 → 西湖音乐喷泉（19:00/20:00）

#### 第二天（深度文化体验）
上午：灵隐寺（飞来峰石刻造像，游览约2-3小时）
下午：龙井村/梅家坞（品茶）→ 六和塔/钱塘江大桥
> 可选增加中国丝绸博物馆、西湖博物馆

## 特色主题路线

### 亲子游路线
1. **上午**：花港观鱼（喂鱼观鱼，小朋友最爱）→ 苏堤漫步 → 断桥（放风筝）
2. **午餐**：湖滨银泰或外婆家
3. **下午**：西湖游船（自开船体验，30元/半小时）→ 少年宫游乐场
4. **傍晚**：湖滨音乐喷泉

> **提示**：西湖游船（自开船/画舫）适合全家乘坐；白堤、苏堤路况平坦，可推婴儿车。

### 老年人游览路线
1. **上午**：白堤漫步（路面平坦，约1公里）→ 平湖秋月赏景 → 孤山公园
2. **午餐**：楼外楼（孤山店，就近用餐）
3. **下午**：环湖观光车（全程40元，坐着看西湖，招手即停）→ 花港观鱼
4. **傍晚**：湖滨路散步

> **提示**：全程路面平坦，建议乘坐观光车减少步行，各景点均有休息椅和坡道。

### 雨天备选路线
1. 上午：浙江省博物馆孤山馆区（免费）→ 西湖博物馆（免费，南山路）
2. 午餐：河坊街（有遮蔽的步行街）
3. 下午：中国丝绸博物馆（免费）→ 湖滨银泰in77（室内逛街）

> **提示**：杭州年均降雨日约150天，建议随身携带雨具。雨中西湖烟雨朦胧，别有韵味。

## 交通指南

### 地铁

| 线路 | 站点 | 对应景区 |
|------|------|---------|
| 1号线 | 龙翔桥站（C口出步行3分钟到西湖） | 湖滨公园、音乐喷泉（最核心站） |
| 1号线 | 凤起路站 | 断桥、北山街 |
| 1号线 | 定安路站 | 河坊街、南宋御街 |
| 1号线 | 城站站 | 杭州站（城站火车站） |
| 3号线 | 黄龙洞站 | 黄龙洞景区 |
| 3号线 | 龙舟路站 | 西溪湿地 |
| 4号线 | 水澄桥站 | 八卦田遗址、白塔公园 |
| 7号线 | 吴山广场站 | 南山路、柳浪闻莺 |
| 19号线 | 西湖文化广场站 | 西湖文化广场（换乘1号线可达湖滨） |

> **地铁运营时间**：约06:00-23:00，各线路首末班车略有差异。

### 从杭州东站到西湖
- **地铁1号线**（推荐）：杭州东站 → 龙翔桥站，约25-30分钟，票价3-4元
- **打车**：约30-40元，25分钟（不堵车时），节假日建议坐地铁

### 从萧山机场到西湖
- **地铁1号线**（直达）：萧山机场站 → 龙翔桥站，约50-60分钟，票价6-8元
- **地铁19号线+换乘1号线**：机场快线更快，约40-50分钟
- **机场大巴**：武林门/平海路线，约60-70分钟，票价20元
- **打车**：约100-150元，40分钟（不堵车时）

### 从杭州站（城站）到西湖
- **地铁1号线**：城站 → 龙翔桥站，仅2站，约5分钟
- **步行**：出站后向西步行约15-20分钟即可到西湖

### 公交
- 西湖环湖观光巴士（招手即停，覆盖主要景点）
- Y1路：灵隐→西湖→少年宫
- Y2路：城站火车站→西湖→灵隐
- 西湖内环线/外环线公交

### 共享单车
- 西湖周边大量共享单车（哈啰、美团、青桔）投放
- 骑行环湖约1.5小时（约13.5公里），白堤、苏堤部分路段禁骑
- 注意：上满觉陇、龙井等坡道区域不适宜骑行

## 自驾停车指南

### 景区停车注意事项
- **单双号限行**：周末及节假日，西湖景区执行单双号通行规则
- **灵隐区域**：周末节假日8:00-17:00禁止出租车/网约车驶入，需换乘公交接驳线

## 公共卫生间位置

西湖沿线主要景点均设有公共卫生间，分布密集，标识清晰。主要位置包括：
- 断桥入口处、白堤中段、平湖秋月旁
- 孤山公园内、西泠印社入口
- 岳王庙景区内、曲院风荷入口
- 苏堤各桥附近（六桥均有）
- 花港观鱼入口及园内
- 雷峰塔景区内、净慈寺入口
- 湖滨路沿线每约500米一处
- 南山路沿线（柳浪闻莺、长桥公园）

> 卫生间在西湖景区均设有引导标识，也可通过"杭州西湖风景名胜区"微信公众号查询附近卫生间。

## 最佳游览时间
- **春季（3-5月）**：苏堤春晓、桃红柳绿，最佳综合旅游季节
- **夏季（6-8月）**：曲院风荷、赏荷听雨，清晨和傍晚最舒适
- **秋季（9-11月）**：满陇桂雨、平湖秋月，桂花飘香
- **冬季（12-2月）**：断桥残雪、探梅寻香（孤山赏梅），游客较少

> **推荐时段**：工作日早晨（7:00-9:00）人最少；周末和节假日建议早于8:00到达。

## 实用建议
- 建议穿舒适步行鞋（一日游步行约1.5-2万步）
- 随身携带雨具（杭州年均降雨日约150天）
- 提前查看景区实时客流（微信公众号可查）
- 西湖音乐喷泉每日19:00/20:00各一场（湖滨三公园）
- 旺季建议早于8:00到达景区，避开10:00-15:00的人流高峰
- 环湖游建议使用共享单车+观光车+游船组合交通方式
- 夏季注意防暑防晒，冬季注意防寒保暖
