## 核心流程

1. 用户语音输入 → FunASR 转文字（降级: Web Speech API）
2. 文本 → Dify RAG 检索景区知识（降级: 直连 DeepSeek）
3. 检索结果 + 用户问题 → DeepSeek LLM 生成回答
4. 回答文本 → Edge-TTS 合成语音（降级: pyttsx3）
5. 语音播报 + Live2D 口型同步（降级: 静态 PNG + CSS）

## 技术要点

- 纯 CPU 运行，零 GPU 依赖
- 每个外部依赖均有降级方案
- 对话记录持久化到 SQLite
